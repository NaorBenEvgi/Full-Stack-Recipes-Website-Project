get /recipes/search/query/{searchQuery}/amount/{num}
search recipes based on params that send by a query (recipesSearchQuerySearchQueryAmountNumGet)
Serach for recipes by a search query. Will return resuslts from spoonacular API, according to number param, which can be filtered by Cusine, diet, intolerance.

Path parameters

searchQuery (required)
Path Parameter — The recipe search query
num (required)
Path Parameter — The number of results to return (5/10/15)

Query parameters

cuisine (optional)
Query Parameter — The cuisine(s) of the recipes
diet (optional)
Query Parameter — The diet for which the recipes must be suitable
intolerances (optional)
Query Parameter — All recipes returned must not contain ingredients that are not suitable for people with the intolerances entered.

array[RecipePreview]
